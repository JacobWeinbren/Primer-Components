# IconButton Component

The `IconButton` component leverages GitHub's Primer Octicons to render SVG icons. Icons serve as visual symbols representing ideas, objects, or actions.

## Usage

To use the `IconButton` component, load in an icon from [Octicons Svelte](https://github.com/metonym/svelte-octicons) which should match the name of the desired icon from the Primer Octicons library. Optionally, the `className` prop can be used to add custom styling.

## Icon Library

This component utilizes icons from GitHub's Primer Octicons library. Explore the full list of icons and their usage guidelines at:

-   [Primer Octicons Library on GitHub](https://github.com/primer/octicons)
-   [Primer Octicons Usage Guidelines](https://primer.style/components/icon)
-   [Svelte Octicons Library](https://github.com/metonym/svelte-octicons)

## Accessibility

-   It is recommended to add an `aria-label` attribute to your icon for screen readers, enhancing accessibility.
