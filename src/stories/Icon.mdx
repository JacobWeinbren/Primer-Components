# Icon Component

The `Icon` component leverages GitHub's Primer Octicons to render SVG icons. Icons serve as visual symbols representing ideas, objects, or actions.

## Usage

To utilise the `Icon` component, specify the `iconName` prop, which should match the name of the desired icon from the Primer Octicons library. Optionally, the `className` prop can be used to add custom styling.

<Source
	language="html"
	code={`<primer-icon aria-label="alert icon" iconName="alert" className="text-fg-danger" size="lg"/>`}
	dark={false}
/>

## Icon Library

This component utilizes icons from GitHub's Primer Octicons library. Explore the full list of icons and their usage guidelines at:

-   [Primer Octicons Library on GitHub](https://github.com/primer/octicons)
-   [Primer Octicons Usage Guidelines](https://primer.style/components/icon)

## Accessibility

-   It is recommended to add an `aria-label` attribute to your icon for screen readers, enhancing accessibility.
